Welcome — let’s set up your AI key

Progress Tracker uses AI to read the free-text daily reports your reporters send in and to write the accountability brief. Pick whichever provider you already have an account with — you only need one key. Cost is a few cents.

Anthropic — Claude

default model: claude-haiku-4-5 · ~$1/$5 per million tokens

Get a key at console.anthropic.com → API Keys. Your key starts with sk-ant-.

OpenAI — GPT

default model: gpt-5.4-mini · ~$0.75/$4.50 per million tokens

Get a key at platform.openai.com/api-keys. Your key starts with sk-.

Your key stays on this computer in a file called .env. It’s never uploaded to GitHub.
